# Discord Message Cleanup

Private local Chrome extension for deleting your own deletable messages in the current Discord web chat.

## Use

1. Open `chrome://extensions`.
2. Enable Developer mode.
3. Click Load unpacked.
4. Select this folder.
5. Open a Discord channel or DM in `https://discord.com`.
6. Open the extension popup, scan, then delete.

## Notes

- Uses the active Discord web session in your browser.
- Runs locally in the current tab.
- Does not collect, sell, or transmit analytics.
- Automated account activity may risk platform action.

## Package

The production ZIP is generated under `dist/`.
